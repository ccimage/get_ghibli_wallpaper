# get_ghibli_wallpaper
nodejs spyder for ghibli wallpaper
--

依赖 promise ，我用了bluebird，其他也可以, cheerio 爬虫

使用方法  
-- 

1. npm i 安装依赖
2. 修改文件保存的路径 gFilePath 
3. 执行 node index.js 等待完成
